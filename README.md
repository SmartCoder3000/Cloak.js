# Cloak.js
The all in one cloaking library for developers

## Easy use
**Put this in your `<head>` tag**
```html
<script src="https://cdn.jsdelivr.net/gh/System32-Dev/Cloak.js/cloak.js"></script>
```
**To actually use**
```js
var cloaker = new TabCloak({
  url: 'https://google.com/'
});
cloaker.main();
```
